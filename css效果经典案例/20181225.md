## 题目

**怎么在移动端画出 0.5px 或者真正1px的线框？**

## 答案

```css
a {
    position: relative;
}
a:after {
    position: absolute;
    content: "";
    top: 0;
    left: 0;
    box-sizing: border-box;
    border: 1rpx solid #50cc82;
    border-radius: 12px;
    pointer-events: none;
    -webkit-transform-origin: 0 0;
    -ms-transform-origin: 0 0;
    transform-origin: 0 0;
    width: 200%;
    height: 200%;
    -webkit-transform: scale(.5);
    -ms-transform: scale(.5);
    transform: scale(.5);
};
```
