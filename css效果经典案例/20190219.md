## 答案
// html部分
```html
<div class="container">
    <div class="circle circle1"></div>
    <div class="circle circle2"></div>
```

// css部分
```css
*{margin:0;padding:0;font-size:14px;}
body{text-align:center;}
.container{position:relative;width:10rem;height:10rem;background-color:#eee;margin:3rem auto 0;}
.circle{width:1rem;height:1rem;background-color:#f00;border-radius:50%;position:absolute;left:5.5rem;top:7.5rem;-webkit-transform-origin:-0.5rem -1.5rem;transform-origin:-0.5rem -1.5rem;
animation:rotate 3.6s linear infinite;-webkit-animation:rotate 3.6s linear infinite;}
.circle1{}
.circle2{-webkit-animation-delay:.3s;animation-delay:.3s;background-color:#8BC34A;}
.circle3{-webkit-animation-delay:.6s;animation-delay:.6s;background-color:#FFB74D;}
.circle4{-webkit-animation-delay:.9s;animation-delay:.9s;background-color:#5C6BC0;}
.circle5{-webkit-animation-delay:1.2s;animation-delay:1.2s;background-color:#ff6600;}
.circle6{-webkit-animation-delay:1.5s;animation-delay:1.5s;background-color:#040038;}
@-webkit-keyframes rotate{
    0%{transform:rotate(0) scale(1);}
    25%{transform:rotate(180deg) scale(1.5);}
    50%{transform:rotate(360deg) scale(1);}
    75%{transform:rotate(540deg) scale(1.5);}
    100%{transform:rotate(720deg) scale(1);}
}
```
